#####
scurl
#####

secure curl
###########

:Manual section: 1

SYNOPSIS
========

.. code-block:: sh

  scurl [options / URLs]

DESCRIPTION
===========

scurl is a wrapper around curl that enforces TLSv1.3.

OPTIONS
=======

See curl(1).

EXIT STATUS
===========

See curl(1).

EXAMPLES
========

.. code-block:: sh

  scurl https://example.com/

SEE ALSO
========

curl(1)
