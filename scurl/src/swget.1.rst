#####
swget
#####

secure wget
###########

:Manual section: 1

SYNOPSIS
========

.. code-block:: sh

  swget [options / URLs]

DESCRIPTION
===========

swget is a wrapper around wget that enforces TLSv1.3.

OPTIONS
=======

See wget2(1).

EXIT STATUS
===========

See wget2(1).

EXAMPLES
========

.. code-block:: sh

  swget https://example.com/

SEE ALSO
========
