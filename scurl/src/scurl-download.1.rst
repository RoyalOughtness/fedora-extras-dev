##############
scurl-download
##############

download files securely
#######################

:Manual section: 1

SYNOPSIS
========

.. code-block:: sh

  scurl-download [options / URLs]

DESCRIPTION
===========

scurl-download is a wrapper around scurl optimized to download files.

OPTIONS
=======

See curl(1).

EXIT STATUS
===========

See curl(1).

EXAMPLES
========

.. code-block:: sh

  scurl-download https://example.com/

SEE ALSO
========

curl(1) scurl(1)
